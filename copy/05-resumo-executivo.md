## 🔄 FLUXO COMPLETO DO USUÁRIO

```
Tráfego (Meta/Google/Orgânico)
        ↓
[1] LANDING PAGE
    Headline: "A maioria está forçando a vista..."
    CTA: "QUERO FAZER MEU DIAGNÓSTICO AGORA"
        ↓
[2] QUIZ (6 perguntas — barra de progresso visível)
    Pergunta 1 → 2 → 3 → 4 → 5 → 6
        ↓
[3] DELAY DE PERSONALIZAÇÃO (4-6 segundos)
    "Analisando suas respostas..."
        ↓
[4] CAPTURA DE WHATSAPP (compromisso/coerência)
    "Pra onde envio seu resultado completo?"
        ↓
[5] RESULTADO PERSONALIZADO (1 dos 3 perfis)
    Verde / Amarelo / Vermelho
        ↓
[6] GRAND SLAM OFFER
    Exame grátis + Óculos de sol com grau de brinde
        ↓
[7] CTA: WHATSAPP DIRETO COM A ÓTICA
    Mensagem pré-formatada esperando resposta
        ↓
[8] CONVERSÃO: AGENDAMENTO DO EXAME
        ↓
[9] EXAME PRESENCIAL → VENDA DO ÓCULOS + BRINDE
```

## 🎯 MÉTRICAS PROJETADAS (benchmark realista para o nicho)

| Etapa | Conversão Esperada | Comentário |
|---|---|---|
| Landing → Início do Quiz | 35–55% | Headline forte + curiosidade |
| Quiz iniciado → Quiz completo | 60–80% | Barra de progresso ajuda |
| Quiz completo → Captura WA | 50–70% | Compromisso/coerência (já investiu) |
| Captura WA → Resposta do lead | 40–60% | Mensagem boa de abertura |
| Resposta → Agendamento | 30–50% | Atendimento humano + escassez |
| Agendamento → Exame realizado | 60–80% | Lembrete WhatsApp |
| Exame → Venda do óculos | 40–65% | Brinde + atendimento |

**Funil consolidado: ~3–8% de visitante para venda fechada.** Para um nicho local de saúde visual, isso é forte.
