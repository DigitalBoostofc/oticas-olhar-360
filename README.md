# Óticas Olhar 360° — Landing Page

Landing page de captação de leads para a **Óticas Olhar 360°** (Camocim/CE), com quiz interativo de diagnóstico visual em 6 perguntas e handoff direto pro WhatsApp.

## 🚀 Stack

- HTML único autocontido (`index.html`)
- Tailwind CSS via CDN
- Google Fonts (Montserrat + Inter)
- JavaScript vanilla (quiz modal, FAQ accordion, mobile menu, scroll reveal)
- Mobile-first / 100% responsivo
